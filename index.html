Summary: Latitude vs X

The purpose of this project was to analyse how weather changes as you get closer to the equater. To accomplish this analysis we had to first pull data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we used MatPlotslib to Plots various aspects of the weather vs. latitude. Factors we looked at included: temperature, cloudiness, wind speed, and humditiy. This site provides the source data and visualizations created as part of the analysis, as well as explanation and descriptions of any trends and correlations witnessed.
